# Shaoxun Liu Academic Homepage

Upload all files and folders in this directory to the root of your GitHub Pages repository.

- `index.html`: homepage with six selected publications
- `publications.html`: complete publication list
- `style.css`: shared page styles
- `assets/profile.jpg`: profile photo
- `.nojekyll`: disables Jekyll processing

The homepage navigation and the “View All Publications” button open `publications.html`.
